CSS Circle

Pure CSS Animation

Circle Animation

By using CSS border-radius, keyframes to scale, and layer blend modes, I'm going to replicate an animated poster style I saw on Secret Riso Club's Instagram. Mix blend mode is not supported on Internet Explorer and looks best in Firefox.

This CSS experiment was created by Megumi on Aug 3, 2021.
Typefaces: GlyphWorld by Leah Maldonado and Syne by Bonjour Monde.
